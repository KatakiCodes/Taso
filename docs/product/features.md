# Product Context

## Nome

Taso

## Objetivo

Uma aplicação simples para gerenciamento de tarefas.

## Público

Usuários individuais.

## Funcionalidades

- Login
- Cadastro
- CRUD de tarefas
- Categorias
- Prioridade
- Datas
- Pesquisa
- Filtros

## Não faz parte do MVP

- Compartilhamento
- Colaboração
- Notificações
